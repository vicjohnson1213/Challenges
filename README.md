## Sorting Algorithms

### [Alternating Sort]()

Sorts an array such that the result alternates between lowest remaining value and highest remaining value.

```
[1, 2, 3, 4] => [1, 4, 2, 3]
[4, 3, 2, 1] => [1, 4, 2, 3]
[2, 1, 4, 3] => [1, 4, 2, 3]
['a', 'b', 'c', 'd'] => ['a', 'd', 'b', 'c']
[] => []
```
